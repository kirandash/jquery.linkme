#Linkme jQuery Plugin#

**Linkme** is a simple jQuery plugin to attach links to elements

##Version##
0.3.0

##Usage##
    $('span').linkme({
		href : 'http://prolificthemes.com',
		text: 'Click me',
		target: '_blank'
	});

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
